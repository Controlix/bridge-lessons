# Product Mission

## Problem

There is a lack of quality interactive learning resources for people wanting to learn bridge. Many existing resources are either purely text-based tutorials or require paid subscriptions.

## Target Users

Complete beginners who have never played bridge before and want an accessible, free way to learn the basics.

## Solution

An interactive bridge learning website with structured lessons and hands-on exercises that let learners practice with real scenarios, not just read about concepts.

## Scope

This app specifically teaches **Berry's Vijfkaart Hoog** (Berry's 5-card major), a popular Dutch bidding system by Berry Westra. Reference: `agent-os/reference/B5Cboek.pdf`
