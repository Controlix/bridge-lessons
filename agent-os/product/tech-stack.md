# Tech Stack

## Frontend

- Blade templates (Laravel's native templating)
- Vanilla JavaScript for interactivity
- CSS for styling

## Backend

- Laravel (PHP)
- No authentication required

## Database

- None required (content is hardcoded/static)

## Testing

- Pest (PHP testing framework)

## Other

- Deployment: Personal cloud space (shared hosting compatible)
- Local development: Laravel Artisan serve
